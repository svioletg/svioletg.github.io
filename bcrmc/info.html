Joining The Server

  • Make sure you are on 1.20.2. You can play with both vanilla & modded clients, but only client-side mods will load. Server-side mods will simply be ignored.
  • The server is running on Minecraft Java Edition, however Bedrock players can also join thanks to the GeyserMC plugin.
  • If you are on Java, you'll connect via Multiplayer → Direct Connect or Add Server, using the address play.bcr-mc.xyz. If you are on Bedrock, see below - you will only have to do this once!

    • Switch users:

    (Summarized from this article.)
    1. Open the System Settings by selecting the gear icon on the home screen, and scroll down to the Internet section.
    2. Select Internet Settings, select the network you are connected to (there should be a checkmark to the right of it), and select Change Settings.
    3. Change DNS Settings to Manual, then set your Primary DNS to 104.238.130.180, and set your Secondary DNS to 1.1.1.1.
    4. Select Save, and open Minecraft.
    5. At the title screen, choose Play, followed by the Servers tab.
    6. Select anything from the Featured Servers list, and then Join Server.
    7. A new menu will appear - select Connect to a Server. From here, you'll continue from Step 3 of the "Other Bedrock users" section below — make sure that the Add to server list toggle is flipped ON (it should look green) so that you don't have to do this each time.

    Other Bedrock users:

    1. Hit Play, then select the Servers tab.
    2. Scroll to the bottom of the list, and select Add Server.
    3. In the Server Name box, enter whatever you'd like. This will only affect the title of the server for yourself.
    4. In the Server Address box, enter play.bcr-mc.xyz.
    5. In the Port box, enter the number 19132 if it is not there already.
    6. The server will now be available at the bottom of your Servers list, in the future.

Active Datapacks

  • Record Every Statistic
    Records Minecraft statistics to scoreboard objectives that can be used for commands.
  • TreeCapitator
    Allows chopping down a whole tree by breaking one block. Can be configured per player by typing /trigger TreeCapitator. Will use the equivalent amount of tool durability as if you'd used it on every block, you can chop single blocks by sneaking.
  • Vanilla Tweaks
    Various quality-of-life tweaks; see Active Tweaks for individual information and how to activate them.

Active Plugins

  • BetterSleeping
    Allows skipping the night with less than 100% of players in bed.
  • Chunky
    Pre-generates chunks to mitigate server lag.
  • squaremap
    Live map that updates real-time & can be viewed online.
  • EssentialsX
    Various commands like teleport requesting & setting nicknames.
  • EssentialsX Chat
    Add-on for EssentialsX with chat features.
  • FreedomChat
    Disables the chat reporting system introduced in 1.19.
  • GeyserMC
    Plugin allowing Bedrock Edition players to join the server.
  • LuckPerms
    Used for setting & modifying permissions. The server really only has OPs and non-OPs as far as permissions are concerned, but this is required by some plugins.
  • SetHomes
    Use /set home to set a home, and /home to teleport back to it. You can set multiple homes with different names.
  • Scythe
    Allows right-clicking to harvest and replant crops automatically. Use /scythe toggle to turn this on or off.

Recommended Mods

Every mod below works with Fabric, is likely to work with Quilt, and will not work with Forge.
Bedrock Edition players cannot use these mods.
No mods at all are required to join the server, these are just suggestions.

Common dependencies; mods that a large amount of other mods depend on to function correctly.


  • ModMenu
    Provides a menu for configuring settings on individual mods. Most Quilt/Fabric mods either support or require this.

Optimization & performance; helps the game run smoother and faster, reduces lag.

  • Lithium
  • Phosphor
  • Sodium
    The three mods above are very strongly recommended, as they vastly improve the game's performance and do not alter the game otherwise.

Quality of life; makes certain tasks easier, more convenient, generally aims to improve the gameplay experience.

  • Mouse Wheelie
    Adds the ability to move items in and out of inventories with the scroll wheel, or shift-clicking and dragging.
  • Xaero's Minimap
    Adds a configurable minimap to the screen, along with waypoints.
  • Xaero's World Map
    Provides access to a full world map that fills in as you explore.

Cosmetics; purely cosmetic/aesthetic changes to make the game look nice.

  • Continuity
    Enables connected textures for resource packs that support it. Requires Indium if you're also using Sodium.
  • Effective
    A successor to the previously recommended "Illuminations"; adds fireflies, waterfall effects, and more that can be customized/configured to your liking.
  • Iris Shaders
    Adds support for shaders.
  • LambDynamicLights
    Makes items like torches and lanterns emit light in your hand.
