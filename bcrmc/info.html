Joining The Server

  • Make sure you are on 1.19.2. It doesn't matter whether your client is vanilla or modded, but only client-side mods will work on the server. You aren't required to disable other mods, they just won't load.
  • This server is running on Minecraft Java Edition, however by using the GeyserMC plugin, Bedrock players should still be able to join.
  • If you are on Java, simply connect via Multiplayer -> Direct Connect or Add Server, using the address play.bcr-mc.xyz. If you are on Bedrock, see below - you will only have to do this once!

    • If you are on Switch:

    (Summarized from this article.)
    1. Open the System Settings by selecting the gear icon on the home screen, and scroll down to the Internet section.
    2. Select Internet Settings, select the network you are connected to (there should be a checkmark to the right of it), and select Change Settings.
    3. Change DNS Settings to Manual.
    4. Set your Primary DNS to 104.238.130.180, and your Secondary DNS to 1.1.1.1. The latter may end up displaying as 001.001.001.001 - this is normal.
    5. Select Save, and open Minecraft.
    6. Select Play, followed by the Servers tab.
    7. Select anything from the Featured Servers list, and select Join Server.
    8. A new menu will appear - select Connect to a Server. From here, you may continue from Step 3 of the "If you are not on Switch" section â€” make sure that the Add to server list toggle is flipped ON (it should look green) so that you don't have to do this each time.

    If you are not on Switch:

    1. Hit "Play", then select "Servers" tab.
    2. Scroll to the bottom of the list, and select "Add Server".
    3. In the "Server Name" box, enter whatever you'd like. This will only affect the title of the server for yourself.
    4. In the "Server Address" box, enter play.bcr-mc.xyz.
    5. In the "Port" box, enter the number 19132 if it is not there already.
    6. The server will now be available at the bottom of your Servers list, in the future.

Active Datapacks

  • Record Every Statistic
    Records Minecraft statistics to scoreboard objectives that can be used for commands.
  • bcrmc4
    Custom datapack made by me for the server. Mainly includes custom advancements.
  • TreeCapitator
    Allows chopping down a whole tree by breaking one block. Can be configured per player by typing /trigger TreeCapitator. Will use the equivalent amount of tool durability as if you'd used it on every block, you can chop single blocks by sneaking.
  • Vanilla Tweaks
    Various quality-of-life tweaks; see list & how to toggle them here.

Active Plugins

  • BetterSleeping
    Allows skipping the night with less than 100% of players in bed.
  • Chunky
    Pre-generates chunks to mitigate server lag.
  • ClickSort
    Allows players to automatically sort inventories. Default is middle-click sorting & sorting by item ID, see link for options & customization.
  • Dynmap
    Live map that updates real-time & can be viewed online. See here for a list of commands to add markers to the map.
  • EssentialsX
    Various commands like teleport requesting & setting nicknames.
  • EssentialsX Chat
    Add-on for EssentialsX with chat features.
  • FreedomChat
    Disables the chat reporting system introduced in 1.19.
  • GeyserMC
    Plugin allowing Bedrock Edition players to join the server.
  • LuckPerms
    Used for setting & modifying permissions. The server really only has OPs and non-OPs as far as permissions are concerned, but this is required by some plugins.
  • SetHomes
    Use /set home to set a home, and /home to teleport back to it. You can set multiple homes with different names.
  • Scythe
    Allows right-clicking to harvest and replant crops automatically. Use /scythe toggle to turn this on or off.

Recommended Mods

Every mod below works with Fabric, may work with Quilt, and will not work with Forge
Bedrock Edition players cannot install mods

Optimization & performance; helps the game run smoother and faster, reduces lag.

  • Lithium
  • Phosphor
  • Sodium
    Highly recommended, I would suggest installing Fabric purely for Sodium alone, even if you plan to use no other mods.

Quality of life; makes certain tasks easier, more convenient, generally improves the gameplay experience.

  • Mouse Wheelie
    Adds the ability to move items in and out of inventories with the scroll wheel, or shift-clicking and dragging.
  • Xaero's Minimap
    Adds a configurable minimap to the screen, along with waypoints.
  • Xaero's World Map
    Provides access to a full world map that fills in as you explore.

Cosmetics; purely cosmetic/aesthetic changes to make the game look nice.

  • Continuity
    Enables connected textures for resource packs that support it. Requires Indium.
  • Illuminations
    Adds fireflies and other misc glowing particles to the game, like in caves.
  • Iris Shaders
    Adds support for shaders.
  • LambDynamicLights
    Makes items like torches and lanterns emit light in your hand. Requires Indium
