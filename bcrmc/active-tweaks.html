1.19: The Deep Dark Update

Currently Active Tweaks

Crafting

  • Rotten Flesh to Leather
    Smelting rotten flesh in a furnace will yield one leather.
  • Charcoal/Coal to Black Dye
    Black dye can be crafted from coal or charcoal.
  • More Bricks
    Crafting four bricks in a square will yield four brick blocks instead of one.
  • More Bark
    Crafting four logs in a square will yield four bark blocks instead of one.
  • Craftable Name Tags
    Name tags are now craftable.
  • Unpackable Wool
    Wool blocks can be crafted into four string.

Other

  • Fast Leaf Decay
    Naturally generated will decay much faster.
  • Durability Ping
    Sends you a chat message if your tool is below 10% durability. Can be customized or disabled with /trigger duraPing.
  • Armored Elytra
    Drop an elytra and a chestplate onto an anvil to combine them. Do the same onto a grindstone to separate them.
  • Double Shulker Shells
    Shulkers drop two shells.
  • Name Colors
    Name color customization. Use /trigger color to customize.
