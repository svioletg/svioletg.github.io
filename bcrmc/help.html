Glazed Terracotta Patterns

  • Each pattern above is 3x3.
  • Glazed terracotta can be acquired by smelting terracotta of the same color.
  • Patterns can be much bigger and more complex than this, but this is a basic example of the smallest ones.
  • Glazed terracotta patterns will always face the direction you were looking in when you placed them. See the magenta pattern for an example of this.
  • Not every block follows the same placement pattern/order to achieve their designs, and of course it can be subjective, so experiment with the placement yourself.
